#### 2. 📕 新增错题本功能

**功能描述：**
- 做错的题目自动加入错题本
- 支持查看、筛选、管理错题
- 掌握后可手动标记移除

**核心特性：**
- **自动记录**：提交练习时，错题自动存入数据库
- **错题本页面**：新增顶部导航"📕 错题本"标签
- **筛选功能**：
  - 全部错题
  - 未掌握（默认）
  - 已掌握
- **统计展示**：未掌握/已掌握/总计数量
- **操作功能**：
  - 标记为已掌握
  - 标记为未掌握
  - 复习此词（一键加入复习队列）
  - 批量复习错题

**数据库变更：**
```sql
CREATE TABLE IF NOT EXISTS wrong_answers (
  id TEXT PRIMARY KEY,
  vocabulary_id TEXT NOT NULL,
  exercise_id TEXT,
  question_type TEXT,
  user_answer TEXT,
  correct_answer TEXT,
  created_at TEXT,
  solved INTEGER DEFAULT 0,
  solved_at TEXT,
  FOREIGN KEY (vocabulary_id) REFERENCES vocabulary(id)
)
```

**新增 API：**
- `GET /api/wrong-answers?unsolvedOnly=true` - 获取错题列表
- `PUT /api/wrong-answers/:id/solve` - 标记为已掌握
- `DELETE /api/wrong-answers/:id` - 删除错题记录

**首页统计：**
- 新增"待掌握错题"统计卡片（红色）

#### 3. 🎨 优化选词页面交互体验

**问题背景：**
- 之前每选一个词就会滚动到顶部，体验很差
- 词汇多了以后，选词费时间

**优化方案：**

##### 3.1 固定列表高度
- 词汇列表固定高度 300px，可滚动
- 选择时不再跳转/滚动
- 保持用户当前浏览位置

##### 3.2 搜索功能
- 添加搜索框，支持按词汇/释义搜索
- 实时过滤显示
- 只在选择时渲染筛选后的词汇

##### 3.3 快速选择按钮
- **全选/取消**：批量选择/取消当前筛选的词汇
- **最近添加**：一键选择最近添加的 20 个词汇
- **未复习**：一键选择所有未复习的词汇

##### 3.4 性能优化
- 选择词汇时只更新计数，不重新渲染整个列表
- 使用 `filterVocabForSelection()` 局部刷新
- 避免不必要的 DOM 操作

**UI 改进：**
- 词汇卡片优化：词汇加粗，释义灰色小字
- 悬停效果：鼠标悬停时背景变色
- 提示文字：底部添加使用提示

### 🚀 使用说明

#### 使用错题本
1. 完成练习后，查看错题详情
2. 点击"复习错题"按钮，一键加入复习队列
3. 或前往"📕 错题本"标签页查看所有错题
4. 掌握后可点击"标记为已掌握"移除

#### 优化后的选词体验
1. 进入"✏️ 复习"页面
2. 使用搜索框快速定位词汇
3. 或使用"最近添加"/"未复习"快速选择
4. 选择词汇时不再滚动，保持浏览位置
