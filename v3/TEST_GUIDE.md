# 测试指南 - 词汇管理助手 v3 更新

## 快速测试步骤

### 1️⃣ 访问应用
打开浏览器访问：http://localhost:10826

### 2️⃣ 测试错题本功能

#### 步骤：
1. 点击顶部"✏️ 复习"标签
2. 选择题型（如"看释义写词语"）
3. 选择 5-10 个词汇
4. 点击"🚀 开始答题"
5. **故意答错几题**（输入错误答案）
6. 完成所有题目后查看结果

#### 预期结果：
✅ 显示总分（带颜色：绿色≥80，橙色≥60，红色<60）
✅ 显示"答对 X 题"和"做错 X 题"
✅ **红色区域显示所有错题详情**：
   - 题目内容
   - 你的答案（红色）
   - 正确答案（绿色）
✅ 出现"复习错题"按钮

### 3️⃣ 测试错题本页面

#### 步骤：
1. 完成练习且有错题后
2. 点击顶部"📕 错题本"标签
3. 查看错题列表

#### 预期结果：
✅ 显示统计：未掌握/已掌握/总计
✅ 显示三个筛选按钮：全部错题/未掌握/已掌握
✅ 每道错题显示：
   - 词汇和拼音
   - 释义
   - 你的答案（红色）
   - 正确答案（绿色）
   - 题目内容
✅ 操作按钮：
   - "标记为已掌握"（绿色）
   - "复习此词"（蓝色）

#### 测试标记功能：
1. 点击"标记为已掌握"
2. 错题变为灰色，显示"✅ 已掌握"
3. 切换到"已掌握"筛选查看
4. 点击"标记为未掌握"可恢复

### 4️⃣ 测试选词优化

#### 步骤：
1. 点击"✏️ 复习"标签
2. 滚动到"选择词汇"区域
3. 观察列表高度是否固定（300px）
4. 在搜索框输入词汇名称
5. 点击快速选择按钮

#### 预期结果：
✅ 列表固定高度，可滚动
✅ 选择词汇时**不会滚动到顶部**
✅ 搜索框实时过滤词汇
✅ "最近添加"按钮选择最近 20 个词汇
✅ "未复习"按钮选择所有未复习词汇
✅ "全选/取消"只影响当前筛选的词汇

### 5️⃣ 测试首页统计

#### 步骤：
1. 返回首页
2. 查看统计卡片

#### 预期结果：
✅ 显示 4 个统计卡片：
   - 📚 总词汇量（蓝色）
   - ➕ 今日录入（绿色）
   - ✅ 今日复习（橙色）
   - 📕 待掌握错题（红色，新增）

## 常见问题排查

### 问题 1：页面样式没有更新
**解决**：强制刷新浏览器（Ctrl+F5 或 Cmd+Shift+R）

### 问题 2：API 返回错误
**解决**：
```bash
cd /home/admin/app/vocabulary-app/v3
./stop.sh
./start.sh
tail -f server.log
```

### 问题 3：错题本页面不存在
**解决**：清除浏览器缓存，或检查 JS 文件是否更新：
```bash
ls -lh /home/admin/app/vocabulary-app/v3/frontend/assets/index-v3.js
```

### 问题 4：数据库错误
**解决**：检查数据库表是否创建：
```bash
cd /home/admin/app/vocabulary-app/v3/backend
sqlite3 vocabulary.db ".tables"
# 应该看到：vocabulary, exercises, wrong_answers
```
